## Projectbeschrijving
Welkom bij het DevOps-project voor het studiejaar 2025-2026! 

In deze repository wordt de praktijk rondom Continuous Integration (CI) en Continuous Deployment (CD) toegepast. Het project bevat momenteel een Node.js/Express API die is geconfigureerd met een geautomatiseerde CI-pipeline via GitHub Actions.

Deze pipeline valideert automatisch de code door:
- Afhankelijkheden te installeren.
- Linter checks (ESLint) uit te voeren om codekwaliteit te garanderen.
- Geautomatiseerde tests (Jest) te draaien inclusief een testdekkingsrapport.

*Dit is een test-push om de CI/CD workflow te activeren.*
